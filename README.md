**AntiFraud Classifier** — это сервис для детектирования подозрительных транзакций в финансовых потоках в реальном времени.  
Система использует декларативное описание правил в формате **YAML**

Поддерживаются мета-поля:
- `enabled`: включение/выключение правила  
- `priority`: числовой приоритет  
- `tags`: тематические метки правила  
- `scope`: область (`src` / `dst` / `both`)  
- `combine`: логика объединения (`all` / `any`)  
- `negate`: инвертирование результата  
- `window_minutes`, `min_count`: агрегирование по временным окнам  
- `severity`: уровень серьёзности (`info` / `warn` / `high`)  
- `action`: рекомендованное действие (`alert`, `block`, `force_kyc`, ...)

Сервис способен отслеживать подозрительные транзакции в потоке, дополнительно отображая тип неблагонадежной опперации, 
указывая степень подозрительности и предлагая возможные действия
